# Releasing

These are instructions on how to release a new version of **stac-geoparquet**.


1. Determine the next version using [semantic versioning](https://semver.org/)
2. Update [CHANGELOG.md](./CHANGELOG.md) for the new version
3. Open a PR with the CHANGELOG update
4. Once approved, merged the PR
5. Create a [Github release](https://github.com/stac-utils/stac-geoparquet/releases) for the new version
6. Once the release is published, this will trigger a release to [PyPI](https://pypi.org/project/stac-geoparquet/)
